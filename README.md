# Bookstore Management

Bookstore Management is a Python web application for handling bookstore operations such as inventory, users, orders, and admin workflows.

## Tech Stack
- Python
- Flask
- Jinja templates
- SQLite/SQLAlchemy-style data utilities

## Features
- Modular bookstore app structure
- Configurable application setup
- Models, routes, and utility layers
- Test suite for core routes and models

## How to Run
1. Clone the repo
   git clone https://github.com/Itsuniquemee/bookstore_management.git
2. Enter project folder
   cd bookstore_management
3. Install dependencies
   pip install -r requirements.txt
4. Start the app (from package entrypoint)
   python bookstore_inventory/app.py
